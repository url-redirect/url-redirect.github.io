# URL Redirect

A simple URL redirect service for demonstration purposes.

## Usage

Redirect to platform.makerumble.com by adding a `?url=` query parameter:

```
https://url-redirect.github.io/?url=/page/12312312312
```

This will redirect to:
```
https://platform.makerumble.com/page/12312312312
```
